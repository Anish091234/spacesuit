# 3D Models

Place your exported 3D models in this folder.

## Expected Files:
- `helmet.glb` - Helmet model
- `legs.glb` (or `thigh.glb`) - Leg/thigh model
- `boots.glb` - Boots model

## Export Settings from Shapr3D:
- Format: GLB (GLTF Binary) - preferred
- Units: Meters
- Include textures/materials: Yes
- Optimize geometry: Yes

## File Size:
- Try to keep each model under 5MB for optimal web performance
- Smaller is better!

Once you add your models here, they will be automatically loaded into the 3D scenes on the website.
